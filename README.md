## SIMPLE QUIZ APP USING REACT AND TYPESCRIPT (Using Open Trivia API)

### Deployed Application:
https://tsquiz.herokuapp.com/

### Project Setup

```javascript
1. Clone the repo
2. cd typescript-react-quiz-app
3. npm install
4. npm start
5. The app should open on localhost: 3000
```

### Tech Stack:

- React.js
- Typescript
- Styled Components
